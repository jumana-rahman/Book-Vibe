# 📚 Book Vibe

A modern React-based web application that allows users to manage their reading activities by organizing books into a Read List and a Wishlist. The application uses client-side routing and persistent storage to provide a smooth user experience.


## ✨ Key Features

- 📖 Browse and view book details
- ✅ Mark books as Read
- ❤️ Add books to Wishlist
- 🔀 Client-side routing using React Router
- 📑 Tab-based interface for easy navigation
- 💾 Persistent data storage using Local Storage
- ⚡ Fast and responsive UI

## 🛠️ Tech Stack

- Frontend: React.js
- Routing: React Router
- State Management: Context API
- Storage: Browser Local Storage
- Styling: (Add Tailwind / CSS / DaisyUI if used)

## ⚙️ Functionality Overview

- Users can explore a list of books from the homepage
- Each book has a detailed view page
- Users can:
- Add books to Read List
- Add books to Wishlist
- Selected books are stored in Local Storage
- Tabs dynamically display:
-- Read books
-- Wishlist items
